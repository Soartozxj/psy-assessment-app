### 字段说明

| 字段         | 说明                             | 示例                                               |
| ------------ | -------------------------------- | -------------------------------------------------- |
| **文件名称** | 文档或文件的完整名称（含扩展名） | `Skills架构改造-架构核心规范.md`                   |
| **文件路径** | 相对于项目根目录的路径           | `/.codebuddy/rules/Skills架构改造-架构核心规范.md` |
| **文档类型** | 文档的分类类型                   | 规范文档、指导文档、代码文件、配置文档、测试文档   |
| **负责人**   | 主要负责该文档的人员             | rich                                               |
| **状态**     | 文档当前状态                     | 草稿、待审核、已发布、已归档、需更新               |
| **版本**     | 文档版本号                       | v1.0.0                                             |
| **最后更新** | 最后更新日期                     | 2026-05-29                                         |
| **用途说明** | 文档的主要用途和内容简介         | 定义架构红线和技术栈约束                           |
| **依赖关系** | 该文档依赖的其他文档             | 无、依赖《规则索引》                               |
| **备注**     | 其他需要说明的信息               | 核心规范，禁止修改                                 |
