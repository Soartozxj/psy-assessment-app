### 1.2 角色职责分工

| 角色                      | 人  | 职责                                                       |
| ------------------------- | --- | ---------------------------------------------------------- |
| **Content Creator（我）** | 🤖  | 内容创作、排版美化、发布执行、数据采集、分析报告、优化迭代 |
| **审核人（你）**          | 👤  | 内容审阅、方向把控、最终批准                               |
| **数据源（各平台）**      | 🔗  | 曝光/互动/转化数据反馈                                     |

### 1.3 沟通方式

```
日常协作：在本会话中完成所有操作
审核方式：我推送内容 → 你给出"通过"/"修改意见" → 我修改 → 再次提交
数据汇报：每周五自动生成本周数据报告，推送给你审阅
```

## 2. 周度运营流程（详细版）

### 2.1 时间线全景

```
周一 09:00 ──── 本周内容批量创作
     │
     ├── 根据上周数据，确定本周选题
     ├── 创作 5-8 篇待审内容
     └── 推送给你审核

周二-周三 ──── 审核轮次
     │
     ├── 你审阅内容，给出反馈
     │   └─ ✅ 直接通过 → 进入发布队列
     │   └─ ✏️ 需修改 → 我修改后再次提交
     └── 定稿后进入发布排期

周四-周五 ──── 发布执行
     │
     ├── 按平台最佳发布时间发布
     ├── 采集发布后 24h 数据
     └── 录入数据追踪表

周五 18:00 ──── 本周数据复盘
     │
     ├── 汇总本周所有内容数据
     ├── 分析成功/失败原因
     ├── 调整下周策略
     └── 输出周报
```

### 2.2 每周产出标准

| 内容类型             | 每周数量   | 时长/字数  | 制作周期          |
| -------------------- | ---------- | ---------- | ----------------- |
| 📝 科普长图文        | 1-2篇      | 800-2000字 | 2-3小时           |
| 🎬 短视频脚本        | 2-3条      | 15-60秒    | 1小时/条          |
| 🖼️ 信息图/轮播图文案 | 1-2组      | 5-8页/组   | 1-2小时           |
| 💬 社交短文          | 2-3条      | 50-200字   | 30分钟/条         |
| 🧪 轻测试/互动       | 1个        | 3-5题      | 1-2小时           |
| **合计**             | **7-11篇** | -          | **约8-12小时/周** |

### 2.3 内容来源矩阵

```
每周选题从哪里来？
     │
     ├── 40% 数据驱动：上周高互动选题的延伸
     ├── 25% 热点驱动：本周社会热点/心理月历节点
     ├── 20% 用户驱动：评论区/社群高频问题
     └── 15% 产品驱动：新功能/新量表发布
```

## 3. 内容创作标准

### 3.1 内容质量门禁（每篇内容必须通过）

```
□ 钩子检查：前3秒/前3行是否能抓住注意力？
□ 信息密度：用户读完后是否获得新认知？
□ 情感共鸣：目标用户是否感同身受？
□ 品牌关联：是否自然连接星蓝心镜产品？
□ 行动引导：是否有清晰的下一步操作指引？
□ 视觉适配：排版/配图是否符合平台规范？
□ 合规检查：不涉及医疗诊断、不制造焦虑、不引发恐慌
```

## 4. 审核与发布流程

### 4.1 审核流程

```
┌─────────────────────────────────────────────────────────┐
│                    审核工作流                             │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  我提交内容 → 你审阅 → ┬→ ✅ 通过 → 进入发布队列         │
│                        │                                 │
│                        └→ ✏️ 修改意见 → 我修改 → 再次提交 │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 4.2 审核回复格式

你只需要回复简单的指令即可：

| 你的回复               | 含义         | 我的操作             |
| ---------------------- | ------------ | -------------------- |
| `✅ 通过`              | 内容没问题   | 进入发布排期         |
| `✏️ 第x篇改一下：xxx`  | 有修改意见   | 按意见修改后再次提交 |
| `❌ 这篇不行，替换`    | 选题方向不对 | 换选题重新创作       |
| `🔄 整体方向调整：xxx` | 策略层面调整 | 调整后续所有内容方向 |

### 4.3 发布规范

#### 各平台最佳发布时间

| 平台           | 最佳发布时间                                  | 频次      |
| -------------- | --------------------------------------------- | --------- |
| **微信公众号** | 周一至周五 12:00-13:00 或 20:00-21:00         | 每周2-3篇 |
| **小红书**     | 工作日 12:00, 18:00, 21:00；周末 10:00, 20:00 | 每日1篇   |
| **抖音**       | 周一二四 19:00-21:00；周末 16:00-20:00        | 每周3-5条 |
| **知乎**       | 工作日 10:00, 15:00, 21:00                    | 每周2-3条 |
| **B站**        | 周五六日 18:00-20:00                          | 每周1-2条 |
| **视频号**     | 工作日晚间 19:00-22:00                        | 每周3-4条 |

#### 发布检查清单

```
□ 标题检查：标题是否吸引点击？是否包含关键词？
□ 封面检查：封面图是否清晰吸引人？
□ 标签检查：是否添加了正确的话题标签？
□ 链接检查：转化链接/二维码是否正确？
□ CTA检查：用户知道下一步应该做什么吗？
□ 排版检查：是否有错别字/格式错误？
```

## 6. 优化迭代机制

### 6.1 迭代周期

```
每日微调 ─→ 每周小迭代 ─→ 每月大迭代
   │            │              │
   ▼            ▼              ▼
 发布时间调整   选题方向优化    内容支柱调整
 标题A/B测试   内容形式调整    平台策略调整
 标签优化      质量门禁更新    用户画像刷新
```

### 6.2 数据驱动的选题优化

```
从本周数据中提取TOP3爆款 → 分析爆款共性：
                              ├── 选题方向？
                              ├── 内容格式？
                              ├── 情绪基调？
                              ├── 发布时段？
                              └── 标题结构？
                              │
                              ▼
                     下周围绕爆款因素
                     生成3-5个延伸选题
```

### 6.3 A/B测试机制

| 测试维度     | 测试内容            | 样本量     | 评估周期 |
| ------------ | ------------------- | ---------- | -------- |
| 标题测试     | 同一内容2个标题版本 | 各2000曝光 | 24h      |
| 封面测试     | 不同封面风格        | 各1000曝光 | 24h      |
| 发布时间测试 | 不同时段发布        | 各500曝光  | 48h      |
| CTA测试      | 不同行动引导语      | 各500曝光  | 48h      |
