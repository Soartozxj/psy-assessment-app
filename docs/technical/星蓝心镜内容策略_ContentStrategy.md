## 1. 策略概览

### 1.1 核心目标

| 目标维度     | 具体指标                                    | 时间框架 |
| ------------ | ------------------------------------------- | -------- |
| **品牌认知** | 建立「星蓝心镜 = 专业AI心理测评」的品牌心智 | 6个月    |
| **用户增长** | 月活跃用户增长 200%，新用户注册提升 150%    | 3个月    |
| **内容触达** | 全平台月均内容曝光量达到 50 万+             | 3个月    |
| **转化提升** | 内容驱动的测评完成率提升 40%                | 6个月    |
| **信任建设** | 内容互动率（评论/分享/收藏）月均增长 20%    | 持续     |

### 1.2 内容定位

```
我们不是冷冰冰的测评工具，而是每个人身边的AI心理陪伴者。
内容传递的是：专业 × 温暖 × 洞察 × 行动
```

- **专业**（可信度）：心理学证据、测评科学、AI技术原理
- **温暖**（亲近感）：故事化表达、真实案例、陪伴式语言
- **洞察**（价值感）：自我认知、行为模式、成长启发
- **行动**（转化力）：可执行的建议、下一步引导、测评入口

### 1.3 品牌声音规范

| 维度     | 规范                                     | 示例                                   |
| -------- | ---------------------------------------- | -------------------------------------- |
| 语调     | 专业但不学术，温暖但不煽情               | "你的情绪有规律可循，让我们一起来发现" |
| 人称     | 第一人称复数「我们」+ 第二人称「你」     | "我们为你准备了一个小测试"             |
| 词语风格 | 避免恐吓式语言，用积极心理学视角         | 用「探索内在」替代「诊断问题」         |
| 长度     | 社交短内容 50-200字，深度内容 800-2000字 | 灵活适配平台特性                       |

### 4.2 内容模板与格式规范

#### 📝 长图文（800-2000字）

**适用场景**：科普深度文、测评结果解读、技术原理

```
【结构模板】
标题：痛点共鸣 + 价值承诺（例：「总在深夜emo？可能是你的"情绪肌肉"需要锻炼」）
导语：一个场景/问题引发共鸣（100字）
正文：
  - 现象描述（200字）：用户熟悉的场景
  - 科学解释（400字）：心理学原理+研究数据
  - 自测指引（200字）：引导到星蓝心镜测评
  - 行动建议（300字）：可执行的改善方法
结语：温暖收尾+CTA引导
配图：3-5张信息图/数据图/场景图
```

#### 🎬 短视频（15-60秒）

**适用场景**：抖音/小红书/视频号

```
【脚本模板 — 30秒版】
0-3s   钩子：抛出反常识/痛点问题（"你以为你失眠是因为压力大？"）
3-10s  信息：一个心理学知识点
10-20s 互动："来测测你的真实焦虑水平" + 引导
20-25s 展示：测评界面/结果截图
25-30s CTA："打开星蓝心镜，输入暗号XXX免费测"

【视觉规范】
- 封面：大字标题 + 情绪化表情/场景
- 字幕：全程配中文字幕（白字黑边）
- 品牌露出：水印logo出现在5-7秒位置
- 竖屏 9:16
```

#### 🖼️ 信息图（单图/轮播）

**适用场景**：小红书/公众号/知乎

```
【内容规范】
- 尺寸：小红书 1242×1660px（3:4），公众号 900×500px
- 品牌色：星蓝 #3B82F6 + 温暖橙 #F97316
- 数据来源标注："数据来自星蓝心镜×x份测评样本"
- 底部统一CTA：扫码/搜索「星蓝心镜」
```

#### 🎙️ 播客/音频（15-30分钟）

**适用场景**：小宇宙/Apple Podcasts/喜马拉雅

```
【选题方向】
- 「心镜漫谈」系列：每月2期，邀请心理咨询师/用户对谈
- 「AI与人心」系列：技术+心理的跨界对话
- 「测评故事」系列：真实用户的测评体验分享
```

#### 🧪 互动测试/H5

**适用场景**：小程序/H5页面/社交裂变

```
【设计原则】
- 题目数：3-5题（轻量），10-15题（深度）
- 结果类型：3-4种结果类型，每种配定制描述+分享卡片
- 分享诱饵：结果海报自动生成，带小程序码
- 转化设计：完整版引导 → 星蓝心镜深度测评
```
