## 一、项目概述

### 1.1 项目背景

星蓝心镜微信小程序已完成核心测评功能，包括：
- ✅ 量表列表与搜索
- ✅ 测评答题与计分
- ✅ 结果展示与AI诊断
- ✅ 历史记录管理
- ✅ 用户中心

**下一步目标**：新增"情绪日记"与"冥想"两大核心功能，提升用户留存与活跃度。

### 1.2 核心目标

1. **情绪日记**：帮助用户记录每日情绪状态，追踪情绪变化趋势
2. **冥想**：提供冥想音频播放，辅助用户放松身心
3. **用户体验**：保持现有设计风格，提供流畅的交互体验

#### 数据结构

**日记记录（diary_entries）**
```sql
CREATE TABLE diary_entries (
  id BIGINT UNSIGNED NOT NULL AUTO_INCREMENT,
  openid VARCHAR(64) NOT NULL COMMENT '用户openid',
  mood_score TINYINT UNSIGNED NOT NULL COMMENT '情绪评分1-5',
  mood_emoji VARCHAR(16) DEFAULT '' COMMENT '情绪emoji',
  content TEXT DEFAULT NULL COMMENT '文字描述',
  related_assessment_id VARCHAR(64) DEFAULT NULL COMMENT '关联的测评记录ID',
  created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (id),
  KEY idx_openid (openid),
  KEY idx_created_at (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='情绪日记记录表';
```

**API接口设计**
- `POST /api/diary` - 创建日记记录

- `DELETE /api/diary/:id` - 删除日记记录

#### 数据结构

**冥想音频（meditation_audios）**
```sql
CREATE TABLE meditation_audios (
  id BIGINT UNSIGNED NOT NULL AUTO_INCREMENT,
  title VARCHAR(128) NOT NULL COMMENT '音频标题',
  description TEXT DEFAULT NULL COMMENT '音频描述',
  category VARCHAR(64) NOT NULL DEFAULT 'sleep' COMMENT '分类：sleep/focus/relax',
  audio_url VARCHAR(255) NOT NULL COMMENT '音频URL',
  duration INT UNSIGNED NOT NULL DEFAULT 0 COMMENT '时长（秒）',
  cover_url VARCHAR(255) DEFAULT NULL COMMENT '封面图URL',
  play_count INT UNSIGNED NOT NULL DEFAULT 0 COMMENT '播放次数',
  is_active TINYINT(1) NOT NULL DEFAULT 1 COMMENT '是否启用',
  sort_order INT NOT NULL DEFAULT 0 COMMENT '排序权重',
  created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
  PRIMARY KEY (id),
  KEY idx_category (category),
  KEY idx_active_sort (is_active, sort_order)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='冥想音频表';
```

**冥想记录（meditation_records）**
```sql
CREATE TABLE meditation_records (
  id BIGINT UNSIGNED NOT NULL AUTO_INCREMENT,
  openid VARCHAR(64) NOT NULL COMMENT '用户openid',
  audio_id BIGINT UNSIGNED NOT NULL COMMENT '音频ID',
  duration INT UNSIGNED NOT NULL DEFAULT 0 COMMENT '实际冥想时长（秒）',
  completed TINYINT(1) NOT NULL DEFAULT 0 COMMENT '是否完成',
  created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
  PRIMARY KEY (id),
  KEY idx_openid (openid),
  KEY idx_created_at (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='冥想记录表';
```

**API接口设计**
- `GET /api/meditation/audios?category=sleep` - 获取冥想音频列表

- `GET /api/meditation/history` - 获取冥想历史

### 3.3 关键技术点

#### 3.3.1 情绪日记 - 星星评分组件

**实现方案**：自定义组件 `mood-rating`

```javascript
// components/mood-rating/mood-rating.js
Component({
  properties: {
    value: { type: Number, value: 0 },
    max: { type: Number, value: 5 }
  },
  methods: {
    onTapStar(e) {
      const value = parseInt(e.currentTarget.dataset.value);
      this.setData({ value });
      this.triggerEvent('change', { value });
    }
  }
});
```

**使用示例**
```xml
<mood-rating value="{{moodScore}}" bind:change="onMoodChange" />
```

#### 3.3.2 冥想播放器 - 后台音频播放

**实现方案**：使用 `wx.createInnerAudioContext()` + 后台播放模式

```javascript
// utils/meditation-helper.js
const audioContext = wx.createInnerAudioContext();
audioContext.obeyMuteSwitch = false; // 不受静音影响

function playMeditation(url) {
  audioContext.src = url;
  audioContext.play();
  
  // 监听播放进度
  audioContext.onTimeUpdate(() => {
    const progress = audioContext.currentTime / audioContext.duration;
    this.setData({ progress });
  });
  
  // 监听播放结束
  audioContext.onEnded(() => {
    this.onPlayEnd();
  });
}
```

**后台播放配置**（app.json）
```json
{
  "requiredBackgroundModes": ["audio"]
}
```

#### 3.3.3 情绪统计图表

**实现方案**：使用 Canvas 绘制简单图表（避免引入大型图表库）

```javascript
// utils/chart-helper.js
function drawMoodTrend(canvasId, data) {
  const ctx = wx.createCanvasContext(canvasId);
  
  // 绘制折线图
  ctx.beginPath();
  data.forEach((point, index) => {
    const x = 20 + index * 50;
    const y = 100 - point.score * 20;
    if (index === 0) {
      ctx.moveTo(x, y);
    } else {
      ctx.lineTo(x, y);
    }
  });
  ctx.stroke();
  
  ctx.draw();
}
```
