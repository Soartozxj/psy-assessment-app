### 1.2 启动本地服务器

#### 方法一：使用 Python（推荐）

```bash
# 进入项目目录
cd /Users/rich/WorkBuddy/20260407113106/mini-app-h5/backend

# 启动 HTTP 服务器（端口 3100）
python3 -m http.server 3100

# 服务器启动后，访问以下地址：
# http://localhost:3100/admin-legacy.html
```

#### 方法二：使用 Node.js

```bash
# 安装 http-server（如果未安装）
npm install -g http-server

# 进入项目目录
cd /Users/rich/WorkBuddy/20260407113106/mini-app-h5/backend

# 启动服务器（端口 3100）
http-server -p 3100

# 服务器启动后，访问以下地址：
# http://localhost:3100/admin-legacy.html
```
