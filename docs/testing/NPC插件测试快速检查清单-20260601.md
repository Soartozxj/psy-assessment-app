## 🧪 测试4：NPC配置功能测试

### 操作步骤

1. 测试加载NPC列表：

   ```javascript
   const plugin = PluginLoader.get('npc');
   plugin.execute({ action: 'list' }).then((result) => {
     console.log('加载了', result.length, '个NPC');
   });
   ```

2. 测试获取单个NPC：

   ```javascript
   const plugin = PluginLoader.get('npc');
   plugin
     .execute({ action: 'list' })
     .then((npcs) => {
       if (npcs && npcs.length > 0) {
         return plugin.execute({ action: 'get', id: npcs[0].id });
       }
     })
     .then((npc) => {
       console.log('获取到NPC:', npc.name);
     });
   ```

3. 测试创建NPC：

   ```javascript
   const plugin = PluginLoader.get('npc');
   plugin
     .execute({
       action: 'create',
       data: { name: '测试NPC', role: '测试角色', avatar: '🤖' }
     })
     .then((result) => {
       console.log('创建NPC成功:', result.name);
     });
   ```

4. 测试与NPC聊天：
   ```javascript
   const plugin = PluginLoader.get('npc');

     .execute({ action: 'list' })

         });
       }
     })
     .then((response) => {
       console.log('NPC回复:', response.reply);
     });
   ```
