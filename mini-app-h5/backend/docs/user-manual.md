# 📚 用户使用手册

## 📋 概述

本手册详细介绍如何使用**心理评估小程序后台管理系统**的插件系统，包括：

- 插件系统介绍
- 计分规则管理插件使用指南
- NPC配置管理插件使用指南
- 常见问题解答

---

## 🎯 插件系统介绍

### 什么是插件？

插件是后台管理系统的**功能模块**，每个插件负责一个独立的功能领域：

| 插件名称              | 功能描述     | 核心功能                                 |
| --------------------- | ------------ | ---------------------------------------- |
| **scoring-plugin.js** | 计分规则管理 | 管理量表的维度、指标、解释规则和筛查规则 |
| **npc-plugin.js**     | NPC配置管理  | 管理咨询师立绘、背景图和过渡语           |

### 插件优势

✅ **模块化** - 每个功能独立，互不干扰  
✅ **可扩展** - 轻松添加新功能插件  
✅ **易维护** - 问题定位快速，修复简单  
✅ **高性能** - 按需加载，减少资源占用

## 📈 计分规则管理插件使用指南

### 功能概述

计分规则管理插件用于配置心理评估量表的**计分规则**，包括：

1. **维度配置** - 定义计分的维度（如躯体化、强迫症状等）
2. **指标配置** - 定义计分的指标（如总分、阳性项目数等）
3. **解释规则** - 配置分数解释的阈值分档
4. **筛查规则** - 配置多条件筛查判定
5. **模拟测试** - 用模拟数据测试计分规则

### 使用流程

#### Step 1: 选择量表

1. 登录后台管理系统
2. 在左侧菜单点击 **"计分规则"**
3. 在量表列表中选择要配置的量表（如 "SCL-90"）

**示意图**：

```
┌─────────────────────────────────────┐
│  计分规则管理                       │
├─────────────────────────────────────┤
│  选择量表: [SCL-90      ] [确认]   │
│                                     │
│  已选量表: SCL-90                  │
│  量表ID: SCL90                     │
│  题目数量: 90                      │
└─────────────────────────────────────┘
```

#### Step 2: 配置维度

**什么是维度？**  
维度是心理评估的**计分方面**，如 SCL-90 包含 10 个维度：

- 躯体化
- 强迫症状
- 人际关系敏感
- 抑郁
- 焦虑
- 敌对
- 恐怖
- 偏执
- 精神病性
- 其他

**添加维度**：

1. 点击 **"添加维度"** 按钮
2. 填写维度信息：
   - **维度名称** - 如 "躯体化"
   - **维度描述** - 如 "包括1、4、12、27、40、42、48、49、52、53、56、58共12题"
   - **题目编号** - 如 "1,4,12,27,40,42,48,49,52,53,56,58"
3. 点击 **"保存"**

**编辑维度**：

1. 在维度列表中找到要编辑的维度
2. 点击 **"编辑"** 按钮
3. 修改维度信息
4. 点击 **"保存"**

**删除维度**：

1. 在维度列表中找到要删除的维度
2. 点击 **"删除"** 按钮
3. 确认删除操作

#### Step 3: 配置指标

**什么是指标？**  
指标是**计分的结果展示**，如：

- 总分
- 阳性项目数
- 阳性症状均分
- 各维度分

**添加指标**：

1. 点击 **"添加指标"** 按钮
2. 填写指标信息：
   - **指标名称** - 如 "总分"
   - **计算公式** - 如 "sum"（求和）、"average"（平均值）、"count"（计数）
   - **指标描述** - 如 "所有题目得分之和"
3. 点击 **"保存"**

**编辑/删除指标** - 操作同维度

#### Step 4: 配置解释规则

**什么是解释规则？**  
解释规则是根据**分数范围**给出相应的**解释文本**，如：

| 分数范围 | 解释文本         |
| -------- | ---------------- |
| 0-1.5    | 心理健康状况良好 |
| 1.5-2.5  | 有轻度心理问题   |
| 2.5-3.5  | 有中度心理问题   |
| 3.5-5.0  | 有重度心理问题   |

**添加解释规则**：

1. 点击 **"添加解释规则"** 按钮
2. 填写规则信息：
   - **指标名称** - 选择要解释的指标（如 "总分"）
   - **最小值** - 如 "0"
   - **最大值** - 如 "1.5"
   - **解释文本** - 如 "心理健康状况良好"
3. 点击 **"保存"**

**编辑/删除解释规则** - 操作同维度

#### Step 5: 配置筛查规则

**什么是筛查规则？**  
筛查规则是根据**多个条件**判定用户是否需要**进一步筛查**，如：

**规则示例**：

- 条件1: 总分 > 160
- 条件2: 或 阳性项目数 > 43
- 条件3: 或 任一因子分 > 2
- **判定**: 需要筛查

**添加筛查规则**：

1. 点击 **"添加筛查规则"** 按钮
2. 填写规则信息：
   - **规则名称** - 如 "SCL-90 筛查规则"
   - **条件列表** - 添加多个条件（指标、运算符、阈值）
   - **逻辑关系** - "AND"（所有条件都满足）或 "OR"（任一条件满足）
   - **判定文本** - 如 "建议进一步评估"
3. 点击 **"保存"**

**编辑/删除筛查规则** - 操作同维度

#### Step 6: 模拟测试

**为什么要模拟测试？**  
在正式使用前，用**模拟数据**测试计分规则，确保配置正确。

**进行模拟测试**：

1. 点击 **"模拟测试"** 按钮
2. 输入模拟数据：
   - 方式1: 随机生成（点击 "随机生成" 按钮）
   - 方式2: 手动输入（在每个题目后输入分数 0-4）
3. 点击 **"开始测试"**
4. 查看测试结果：
   - 各维度得分
   - 各指标得分
   - 解释规则匹配结果
   - 筛查规则判定结果

**示例测试结果**：

```

│  模拟测试结果                       │
├─────────────────────────────────────┤
│  总分: 180                         │
│  阳性项目数: 50                    │
│  阳性症状均分: 2.5                 │
│                                     │
│  维度得分:                          │
│    - 躯体化: 1.8                   │
│    - 强迫症状: 2.2                 │
│    - 抑郁: 2.5                     │
│    - ...                           │
│                                     │
│  解释结果:                          │
│    心理健康状况: 有中度心理问题      │
│                                     │
│  筛查判定:                          │
│    ✅ 建议进一步评估                │
└─────────────────────────────────────┘
```

#### Step 7: 保存配置

完成所有配置后，点击 **"保存配置"** 按钮，保存所有配置到服务器。

**提示**：

- ✅ 保存成功后会显示提示消息
- ⚠️ 记得每次修改后都要保存
- 💡 可以点击 "重置配置" 清除所有修改

## 🎭 NPC配置管理插件使用指南

### 功能概述

NPC配置管理插件用于配置**咨询师大厨**的相关资源，包括：

1. **咨询师立绘管理** - 管理咨询师的头像、立绘图片
2. **背景图管理** - 管理聊天界面的背景图片
3. **过渡语管理** - 管理咨询师的开场白、过渡语
4. **选项管理** - 管理咨询师的身份、风格选项

### 使用流程

#### Step 1: 咨询师立绘管理

**什么是咨询师立绘？**  
咨询师立绘是**咨询师的头像和全身图片**，用于在聊天界面展示。

**添加咨询师**：

1. 在左侧菜单点击 **"NPC配置"**
2. 选择 **"咨询师管理"** 标签页
3. 点击 **"添加咨询师"** 按钮
4. 填写咨询师信息：
   - **咨询师名称** - 如 "张医生"
   - **身份** - 选择 "心理咨询师" 或 "心理医生"
   - **风格** - 选择 "专业型" 或 "亲切型"
   - **描述** - 如 "擅长认知行为疗法"
   - **立绘图片** - 上传咨询师的立绘图片
5. 点击 **"保存"**

**设置默认咨询师**：

1. 在咨询师列表中找到要设置为默认的咨询师
2. 点击 **"设为默认"** 按钮
3. 确认设置操作

**提示**：

- ✅ 默认咨询师会优先展示给用户
- ⚠️ 只能设置一个默认咨询师
- 💡 可以随时更换默认咨询师

**编辑/删除咨询师**：

1. 在咨询师列表中找到要操作的咨询师
2. 点击 **"编辑"** 或 **"删除"** 按钮
3. 完成操作

**筛选咨询师**：

1. 在咨询师列表上方，使用筛选条件：
   - **按身份筛选** - 选择 "心理咨询师" 或 "心理医生"
   - **按风格筛选** - 选择 "专业型" 或 "亲切型"
2. 点击 **"筛选"** 按钮

#### Step 2: 背景图管理

**什么是背景图？**  
背景图是**聊天界面的背景**，可以设置多张，系统会随机或按规则选择。

**添加背景图**：

1. 在 "NPC配置" 页面，选择 **"背景图管理"** 标签页
2. 点击 **"添加背景图"** 按钮
3. 填写背景图信息：
   - **背景名称** - 如 "温馨风格"
   - **背景图片** - 上传背景图片
   - **描述** - 如 "暖色调，给人温暖感"
4. 点击 **"保存"**

**设置默认背景图**：

1. 在背景图列表中找到要设置为默认的背景图
2. 点击 **"设为默认"** 按钮
3. 确认设置操作

**编辑/删除背景图** - 操作同咨询师

#### Step 3: 过渡语管理

**什么是过渡语？**  
过渡语是**咨询师的开场白和过渡话语**，用于引导用户进入咨询状态。

**添加过渡语**：

1. 在 "NPC配置" 页面，选择 **"过渡语管理"** 标签页
2. 点击 **"添加过渡语"** 按钮
3. 填写过渡语信息：
   - **过渡语类型** - 选择 "开场白" 或 "过渡语"
   - **咨询师** - 选择关联的咨询师
   - **过渡语内容** - 如 "你好，我是张医生，请问有什么可以帮到你？"
4. 点击 **"保存"**

**编辑/删除过渡语** - 操作同咨询师

#### Step 4: 选项管理

**什么是选项？**  
选项是**咨询师的身份和风格选项**，用于在用户选择咨询师时展示。

**管理身份选项**：

1. 在 "NPC配置" 页面，选择 **"选项管理"** 标签页
2. 在 "身份选项" 区域，查看现有选项
3. 可以添加、编辑、删除身份选项

**管理风格选项** - 操作同身份选项

#### Step 5: 保存配置

完成所有配置后，点击 **"保存配置"** 按钮，保存所有配置到服务器。

**云端同步**：

- ✅ 配置会自动同步到云端
- ⚠️ 确保网络连接正常
- 💡 可以在多台设备上同步配置

## ❓ 常见问题解答

### 计分规则管理插件

**Q1: 为什么选择量表后没有显示维度配置？**  
**A**: 可能是以下原因：

1. 量表数据未正确加载 - 请刷新页面重试
2. 量表没有配置过维度 - 请先添加维度
3. 网络问题 - 请检查网络连接

**Q2: 添加维度时，题目编号如何填写？**  
**A**: 题目编号用**英文逗号**分隔，如：

- 正确: `1,4,12,27,40,42`
- 错误: `1、4、12、27、40、42`（中文逗号）
- 错误: `1 4 12 27 40 42`（空格分隔）

**Q3: 模拟测试时，分数范围是多少？**  
**A**: SCL-90 的每个题目分数是 **0-4**：

- 0 = 没有
- 1 = 很轻
- 2 = 中等
- 3 = 偏重
- 4 = 严重

**Q4: 保存配置后，数据丢失了？**  
**A**: 可能是以下原因：

1. 服务器保存失败 - 请检查网络连接后重试
2. 浏览器缓存问题 - 请清除缓存后重试
3. 数据格式错误 - 请检查配置数据是否正确

### NPC配置管理插件

**Q1: 上传图片失败？**  
**A**: 可能是以下原因：

1. 图片大小超过限制 - 请压缩图片后重试（建议 < 500KB）
2. 图片格式不支持 - 请使用 JPG、PNG 格式
3. 网络问题 - 请检查网络连接

**Q2: 设置默认咨询师后，没有生效？**  
**A**: 可能是以下原因：

1. 配置未保存 - 请点击 "保存配置" 按钮
2. 缓存问题 - 请清除缓存后重试
3. 云端同步延迟 - 请等待几秒后刷新页面

**Q3: 过渡语没有显示？**  
**A**: 可能是以下原因：

1. 过渡语未关联咨询师 - 请编辑过渡语，选择关联的咨询师
2. 咨询师未启用 - 请检查咨询师状态
3. 缓存问题 - 请清除缓存后重试

**Q4: 云端同步失败？**  
**A**: 可能是以下原因：

1. 网络问题 - 请检查网络连接
2. 服务器问题 - 请联系管理员
3. 数据冲突 - 请先拉取最新配置，再推送本地配置
