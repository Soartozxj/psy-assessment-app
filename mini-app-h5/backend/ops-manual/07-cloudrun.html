云托管 API 服务

7.1 基本信息

项目 值
服务名 psy-api

7.2 API 接口列表

方法 路径 说明
GET / 健康检查
POST /api/submit 提交测评（计分+存历史）
GET /api/history?page=1&pageSize=20 查询历史
DELETE /api/history/:id 删除单条历史
POST /api/ai-diagnose AI 诊断

7.3 部署/更新

  1. 修改 cloudbase-run/index.js
  2. 重新打包 zip
  3. 云托管 → 服务 → 重新部署
  4. 等待构建完成（约 1-2 分钟）
