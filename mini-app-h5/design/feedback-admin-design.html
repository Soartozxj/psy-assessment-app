💬 后台 · 用户评价管理 — 设计稿 v1

侧边栏新增「用户评价」菜单项，支持全部评价查看、筛选排序、详情弹窗、删除管理。数据来源：localStorage 前端评价。

📍 页面位置

后台侧边栏「数据管理」分组下新增菜单项：

  • NEW 💬 用户评价

与「用户管理」「测评记录」同级。

📊 页面结构（从上到下）

区域内容
统计卡4 个指标：总评价数、平均星级、本月新增、待处理差评
筛选栏搜索 + 量表筛选 + 来源筛选 + 星级筛选 + 排序
双栏图表左：星级分布 + 均分
右：热门标签词云
趋势图近 7 天 / 30 天评价数量趋势
评价列表表格形式，含用户/星级/量表/标签/内容/时间/操作
分页每页 20 条，支持翻页

🔍 筛选能力

  • 关键词搜索：模糊匹配评价内容、用户名
  • 量表筛选：下拉选择具体量表或「全部」
  • 来源筛选：我的页面 / 测评记录 / 测评详情
  • 星级筛选：快捷 Chip（1-5 星 + 全部）
  • 排序方式：最新优先 / 最早优先 / 星级最高 / 星级最低

📋 表格列说明

列说明
用户头像 + 匿名名 + ID
评分星级 + 文字标签
关联量表量表名（整体评价显示「--」）
标签用户选择的标签 Chips
评价内容文字评价，超2行折叠
时间提交时间 + 来源标签
操作查看详情 / 删除

🎭 详情弹窗

  • 点击「查看详情」打开模态框
  • 展示完整评价信息
  • 含「删除」和「关闭」操作

💾 数据存储

  • localStorage 键：psy_feedback_list
  • 数组结构，每条含 id/userId/nickname/stars/tags/text/scaleId/scaleName/source/createdAt
  • 后台可读取、筛选、删除，不可编辑用户评价
