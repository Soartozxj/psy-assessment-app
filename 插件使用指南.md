## 📝 简介

### 什么是插件系统？

插件系统是心理评估小程序的模块化架构，允许功能以插件形式动态加载和卸载。

**核心优势：**

- **模块化**: 每个功能独立开发、测试、部署
- **按需加载**: 减少首屏加载时间，提升用户体验
- **双端适配**: 一套代码同时支持H5和小程序
- **易于扩展**: 新增功能只需创建新插件

### 插件分类

| 类型         | 加载策略               | 示例                     | 使用场景           |
| ------------ | ---------------------- | ------------------------ | ------------------ |
| **核心插件** | 预加载（应用启动时）   | auth                     | 用户认证、基础功能 |
| **可选插件** | 按需加载（用户触发时） | scoring, npc, meditation | 高级功能、可选模块 |

## 🔧 核心插件使用

### Auth插件（用户认证）

**功能**: 用户登录、登出、权限验证

**使用示例:**

```javascript
// 1. 获取插件实例（已预加载）
const authPlugin = PluginLoader.get('auth');

// 2. 用户登录
async function userLogin() {
  try {
    const result = await PluginLoader.execute('auth', {
      action: 'login',
      data: {
        username: 'admin',
        password: '123456'
      }
    });

    console.log('登录成功:', result);
    return result;
  } catch (error) {
    console.error('登录失败:', error.message);
    throw error;
  }
}

// 3. 用户登出
async function userLogout() {
  try {
    const result = await PluginLoader.execute('auth', {
      action: 'logout'
    });

    console.log('登出成功:', result);
    return result;
  } catch (error) {
    console.error('登出失败:', error.message);
    throw error;
  }
}

// 4. 检查登录状态
async function checkLoginStatus() {
  try {
    const result = await PluginLoader.execute('auth', {
      action: 'check'
    });

    console.log('登录状态:', result);
    return result;
  } catch (error) {
    console.error('检查失败:', error.message);
    return false;
  }
}
```

**可用Actions:**

| Action        | 参数                     | 返回值                     | 说明         |
| ------------- | ------------------------ | -------------------------- | ------------ |
| `login`       | `{ username, password }` | `{ success, token, user }` | 用户登录     |
| `logout`      | `{}`                     | `{ success }`              | 用户登出     |
| `check`       | `{}`                     | `{ isLoggedIn, user }`     | 检查登录状态 |
| `getUserInfo` | `{}`                     | `{ id, username, role }`   | 获取用户信息 |

## 🔄 可选插件使用

### Scoring插件（计分规则管理）

**功能**: 心理评估量表的计分规则管理

**使用示例:**

```javascript
// 1. 加载插件（按需加载）
async function loadScoringPlugin() {
  try {
    await PluginLoader.load('scoring');
    console.log('Scoring插件加载成功');
  } catch (error) {
    console.error('Scoring插件加载失败:', error.message);
    throw error;
  }
}

// 2. 获取所有计分规则
async function getAllScoringRules() {
  try {
    const rules = await PluginLoader.execute('scoring', {
      action: 'list'
    });

    console.log('计分规则列表:', rules);
    return rules;
  } catch (error) {
    console.error('获取失败:', error.message);
    throw error;
  }
}

// 3. 获取单个计分规则
async function getScoringRule(ruleId) {
  try {
    const rule = await PluginLoader.execute('scoring', {
      action: 'get',
      id: ruleId
    });

    console.log('计分规则详情:', rule);
    return rule;
  } catch (error) {
    console.error('获取失败:', error.message);
    throw error;
  }
}

// 4. 计算得分
async function calculateScore(ruleId, answers) {
  try {
    const result = await PluginLoader.execute('scoring', {
      action: 'calculate',
      id: ruleId,
      answers: answers
    });

    console.log('计算结果:', result);
    return result;
  } catch (error) {
    console.error('计算失败:', error.message);
    throw error;
  }
}
```

**可用Actions:**

| Action      | 参数              | 返回值               | 说明             |
| ----------- | ----------------- | -------------------- | ---------------- |
| `list`      | `{}`              | `Array<Rule>`        | 获取所有计分规则 |
| `get`       | `{ id }`          | `Rule`               | 获取单个计分规则 |
| `create`    | `{ data }`        | `Rule`               | 创建计分规则     |
| `update`    | `{ id, data }`    | `Rule`               | 更新计分规则     |
| `delete`    | `{ id }`          | `{ success }`        | 删除计分规则     |
| `calculate` | `{ id, answers }` | `{ score, details }` | 计算得分         |

## 🛠️ 插件管理

### 查看已加载插件

```javascript
// 获取所有已加载的插件信息
const pluginsInfo = PluginLoader.getPluginsInfo();
console.log('已加载插件:', pluginsInfo);

// 输出示例：
// [
//   { name: 'auth', displayName: 'Auth插件', version: '1.0.0', ... },
//   { name: 'scoring', displayName: 'Scoring插件', version: '1.0.0', ... }
// ]
```

### 手动加载插件

```javascript
// 加载单个插件
async function loadPlugin(pluginName) {
  try {
    const plugin = await PluginLoader.load(pluginName);
    console.log(`插件 ${pluginName} 加载成功:`, plugin);
    return plugin;
  } catch (error) {
    console.error(`插件 ${pluginName} 加载失败:`, error.message);
    throw error;
  }
}

// 使用示例
loadPlugin('meditation');
```

### 手动卸载插件

```javascript
// 卸载单个插件
function unloadPlugin(pluginName) {
  try {
    PluginLoader.unload(pluginName);
    console.log(`插件 ${pluginName} 卸载成功`);
  } catch (error) {
    console.error(`插件 ${pluginName} 卸载失败:`, error.message);
    throw error;
  }
}

// 使用示例
unloadPlugin('scoring');
```

### 重新加载插件

```javascript
// 重新加载插件（先卸载再加载）
async function reloadPlugin(pluginName) {
  try {
    // 卸载
    if (PluginLoader.isLoaded(pluginName)) {
      PluginLoader.unload(pluginName);
    }

    // 重新加载
    const plugin = await PluginLoader.load(pluginName);
    console.log(`插件 ${pluginName} 重新加载成功:`, plugin);
    return plugin;
  } catch (error) {
    console.error(`插件 ${pluginName} 重新加载失败:`, error.message);
    throw error;
  }
}

// 使用示例
reloadPlugin('npc');
```

## ❓ 常见问题

### Q1: 插件加载失败怎么办？

**排查步骤：**

1. **检查插件文件是否存在**

   ```bash
   # 检查可选插件
   ls -la /mini-app-h5/backend/plugins/optional/

   # 检查核心插件
   ls -la /mini-app-h5/backend/plugins/core/
   ```

2. **检查插件类是否正确导出**

   ```javascript
   // 在插件文件末尾必须有：
   window.PluginNamePlugin = PluginNamePlugin;
   ```

3. **查看控制台错误日志**
   - 打开浏览器控制台（F12）
   - 查看红色错误信息
   - 根据错误信息修复问题

### Q2: 插件执行失败怎么办？

**可能原因：**

1. **插件未加载**

   ```javascript
   // 检查插件是否已加载
   if (!PluginLoader.isLoaded('scoring')) {
     await PluginLoader.load('scoring');
   }
   ```

2. **参数错误**

   ```javascript
   // 检查参数格式
   const result = await PluginLoader.execute('scoring', {
     action: 'get',
     id: 'rule-123' // 确保参数正确
   });
   ```

3. **插件内部错误**
   - 查看控制台错误日志
   - 检查插件源码中的`onExecute`方法

### Q3: 如何调试插件？

**调试技巧：**

1. **在插件中添加日志**

   ```javascript
   async onExecute(params) {
     console.log('收到参数:', params);  // 添加这行

     // ... 原有代码
   }
   ```

2. **使用PluginLoader的调试功能**

   ```javascript
   // 在控制台中调用
   PluginLoader._debug();
   ```

3. **检查插件状态**
   ```javascript
   // 查看插件实例
   const plugin = PluginLoader.get('scoring');
   console.log('插件状态:', {
     isInitialized: plugin.isInitialized,
     isDestroyed: plugin.isDestroyed,
     version: plugin.version
   });
   ```

### Q4: 插件在H5和小程序上都能用吗？

**答：是的，只要正确使用Adapter。**

```javascript
// ✅ 正确做法：使用Adapter
// 存储
Adapter.storage.set('key', 'value');

// 网络请求
const result = await Adapter.http.get('/api/data');

// UI提示
this.showToast('提示', 'info');

// ❌ 错误做法：直接使用平台特定API
// H5特定
localStorage.setItem('key', 'value');

// 小程序特定
wx.setStorage({ key: 'key', data: 'value' });
```
