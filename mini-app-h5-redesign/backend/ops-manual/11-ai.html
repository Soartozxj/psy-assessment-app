AI 接口配置

11.1 架构说明

AI 功能有两个调用路径：

路径 1（前端直连）：H5 → shared-data.js 读取 Key → 直连 DashScope
路径 2（代理调用）：H5 → 云托管 API → 云数据库读配置 → 调用 DashScope
  • WebView 模式（小程序内）：主要走路径 2，更安全
  • 独立 Web 端（浏览器）：走路径 1，需要 ai-config.json 有有效 Key

11.2 API Key 管理

位置 配置方式 说明
ai-config.json 手动编辑 构建时注入到 H5 前端
shared-data.js build-deploy.py 注入 线上运行的配置
云数据库 config.ai_config 后台或控制台 云托管 API 读取
DashScope 控制台 在线管理 管理 Key / 查看用量
