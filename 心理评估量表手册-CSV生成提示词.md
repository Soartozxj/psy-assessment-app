一、处理规则

【0】原文忠实（最高优先级，违反即不合格）

⚠️ 题目内容和选项文本必须逐字照搬手册原文，这是一条不可违反的硬性规则。

具体要求：
- 题目内容：必须与手册原文中的文字完全一致，一个字都不能改
- 选项文本：必须与手册原文中的选项文字完全一致
- 允许的操作：删除题号前缀（如手册写"1.用温和友好的语气与我说话"，提取为"用温和友好的语气与我说话"）；删除题号后的空格和多余标点
- 严禁的操作：改写、意译、简化、润色、替换同义词、用自己的话重述、将口语改为书面语、将长句拆为短句、合并相邻题目

错误示例（以下行为全部禁止）：
  手册原文：不想让我长大 → ❌ 不可改为"使我感到不自在"
  手册原文：试图控制我做的每一件事 → ❌ 不可改为"使我难堪或当众受辱"
  手册原文：经常对我微笑 → ❌ 不可改为"让我感到在家是个受欢迎的成员"
  手册原文：似乎不明白我想要什么 → ❌ 不可改为"信任我"
  手册原文：让我觉得自己是可有可无的 → ❌ 不可改为"对我的私事不感兴趣"
  手册原文：从不夸奖我 → ❌ 不可改为"对我很亲切"

如果你发现手册中的某个题目措辞有语病或不够通顺，也必须原文照搬，不要"修正"它。量表题目是经过心理测量学检验的标准化表述，任何修改都会破坏量表信效度。

【1】多量表识别

手册中可能包含多张量表，常见标识："附："、"附1："、"附2："、"附录一"、"附录二"、"量表一"、"量表二"、或章节标题中的量表名称。

处理方式：
- 逐一识别，每张量表生成一组连续数据行
- 所有量表共用一个表头（只出现一次）
- 每张量表第 1 行填写基本信息，其余题目行留空
- 摘要中说明：共识别到 N 张量表：[名称列表]

【2】版本确认

若手册含多个版本（原版/修订版/简版）且我未指明版本，在摘要中提醒：⚠️ 手册含多个版本，本次基于[XX版本，共N题]，请确认。然后按其中一个版本输出。

【3】题目数量

若我在对话中注明了题目数，严格按该数量提取，不合并不扩展。

【4】反向计分与分值方向

分值方向的核心原则：正向表述的题目，选择"最符合/最严重"应得最高分；反向计分题翻转后也应满足这一原则。

第一步：判断题目表述方向
- 正向表述：题目描述的是"好的/ desirable/ 期望的"行为、品质或感受。选"非常符合"应得最高分，分值递增（如 0,1,2,3）
  常见正向关键词：疼爱、亲切、温和、友好、关心、了解、信任、鼓励、支持、帮助、表扬、重视、喜欢、耐心
- 负向表述：题目描述的是"不好的/ undesirable/ 不期望的"行为、品质或感受。这是反向计分题，需要翻转分值
  常见负向关键词：冷漠、漠不关心、缺乏、没有、不自在、难堪、控制、干涉、担心过度
- 判断方法：问自己"如果一个人在这道题上选了'非常符合'，这个人的状况是好的还是不好的？"好的 → 正向；不好的 → 负向（反向计分题）

第二步：仅对负向表述（反向计分题）翻转分值，翻转公式：新分值 = 最小值 + 最大值 - 原始分值

示例1（正向题，不翻转）：题目"对我很亲切"，分值 0,1,2,3。选"非常符合"说明父母很亲切，应得 3 分（最高分）→ 递增，不翻转 ✅
示例2（正向题，不翻转）：题目"给我很多表扬"，分值 0,1,2,3。选"非常符合"说明父母常表扬，应得 3 分 → 不翻转 ✅
示例3（反向题，需翻转）：题目"对我漠不关心"，原始分值 0,1,2,3 → 翻转后填 3,2,1,0。选"非常符合"说明父母漠不关心，应得 0 分（最低分）→ 需要翻转 ✅
示例4（反向题，需翻转）：题目"使我感到什么事都做不好"，原始分值 0,1,2,3 → 翻转后填 3,2,1,0
示例5（⚠️ 常见错误）：题目"用温和友好的语气与我说话"是正面表述（"温和友好"=好品质），不应翻转。如果翻转成 3,2,1,0 就错了，应该是 0,1,2,3
示例6（⚠️ 常见错误）：题目"对我很亲切"是正面表述（"亲切"=好品质），不应翻转。翻转成 3,2,1,0 就错了

第三步：所有题目的"反向计分"列统一填"否"（因为翻转已由你预处理完成）

注意：仅翻转手册原文中明确标注或根据表述方向明确判断为反向计分的题目。不要翻转正向表述的题目。

【5】异常检测（必须执行）

逐一检查以下 6 类异常，发现任何一项都必须在摘要中报告：

异常类型        判定条件                          处理方式
─────────────────────────────────────────────────────────────────
编号缺失        题号不连续（如 5 后直接 7）      跳过缺失题（不编造），报告中说明位置
编号重复        同一题号出现多次                  后续重编为顺序编号，报告
编号错乱        题号顺序不对但内容完整            按内容顺序重编 1,2,3...，列出原→新映射
内容残缺        有题号但文字缺失或不完整          如实输出现有内容，报告
选项缺失        题目存在但选项不完整              如实输出，空缺留空，报告
维度不明        手册有维度列表但部分题无法归属    按内容推断，列出不确定的题号

报告格式示例：
⚠️ [量表名称] 题目异常：
  - 缺失：第 7、13 题在原文中未找到，已跳过
  - 重复：第 9 题出现两次，第二次重编为第 10 题
  - 内容残缺：第 12 题仅有选项无题目文字

二、CSV 格式规范

【表头】

固定顺序如下，但选项列数按实际最多选项数决定（不固定为 5 个）：

量表名称,量表简称,量表编码,所属分类,图标,主题色,预计时长(分钟),适用人群,量表描述,指导语,标签,状态,排序权重,题号,题目内容,维度,反向计分,选项A文本,选项A分值,选项B文本,选项B分值,[...按需继续到选项Z]

规则：
- 若最多只有 2 个选项，表头到"选项B分值"即止
- 若最多有 7 个选项，表头延伸到"选项G分值"
- 以所有量表所有题目中最多的选项数确定总列数

【字段说明】

列名              必填  规则
─────────────────────────────────────────────────────────────────
量表名称          是    完整名称，如"PHQ-9 抑郁症筛查量表"
量表简称          否    缩写，如"PHQ-9"
量表编码          是    仅字母+数字+下划线，全大写，如"PHQ9"、"FACES_CV"
所属分类          是    只能填以下值之一：anxiety / depression / personality / comprehensive / stress / unknown。不要自行编造其他值（如 family、children、sleep 等）
图标              否    单个 emoji，默认 📋
主题色            否    十六进制，如"#4A90D9"，默认蓝色
预计时长(分钟)    否    数字
适用人群          否    文本
量表描述          否    如含英文逗号须用英文双引号括起
指导语            否    如含英文逗号须用英文双引号括起
标签              否    中文逗号分隔
状态              否    1（上架）或 2（草稿）
排序权重          否    数字，越大越靠前
题号              是    从 1 开始的整数
题目内容          是    如含英文逗号须用英文双引号括起
维度              否    维度/因子名称，留空归为未分类
反向计分          固定  填"否"（分值已预处理翻转）
选项X文本         是    第 X 个选项文字
选项X分值         是    第 X 个选项分数，反向题已翻转

【行规则】

- 每行 = 一道题
- 量表基本信息（前 13 列）只在每张量表的第 1 题行填写，其余行留空（逗号占位）
- 多张量表连续输出，不重复表头

【引号转义】

- ⚠️ 只在字段内容中包含英文逗号（,）、英文双引号（"）或换行符时，才用英文双引号括起该字段。不含这些字符的字段**不要加双引号**。
- 错误示例：用温和友好的语气与我说话 → """用温和友好的语气与我说话"""（❌ 不含英文逗号，不需要双引号）
- 正确示例：用温和友好的语气与我说话 → 用温和友好的语气与我说话（✅ 直接写）
- 正确示例：健康, 幸福 → "健康, 幸福"（✅ 含英文逗号，需要双引号）
- 正确示例：他说"好" → "他说""好"""（✅ 含英文双引号，需要转义）
- 中文逗号（，）和中文引号（""、''）不需要处理，不需要加英文双引号

【样式建议】

颜色：焦虑 #F5A623 抑郁 #5B8DEF 人格 #7ED321 综合 #4A90D9 家庭 #C47EDB
图标：焦虑😰 抑郁💙 人格🧠 综合🧬 家庭👨‍👩‍👧 青少年🌱 睡眠😴 创伤⚡ 认知🔬 其他📋

## 使用流程

1. 复制上方两个 `============` 之间的全部内容
2. 打开任意 AI 对话（ChatGPT / Claude / 通义千问等）
3. **在提示词末尾的"[量表手册原文在下方]"后面，直接粘贴手册原文**
4. ⚠️ **主动补充关键信息**（可选但推荐，写在手册原文后面）：
   - 文件中量表数量和名称（如"文件中含 PHQ-9 和 GAD-7 两张量表"）
   - 每张量表的实际题目数（如"PHQ-9 共 9 题，GAD-7 共 7 题"）
   - 反向计分题目编号（如"RSES 第3、5、8、10题为反向计分"）
   - 期望的量表分类值（如"PHQ-9 分类填 depression"）
5. **一次性发送**（提示词 + 手册原文 + 补充信息一起粘贴发送）
6. AI 先输出摘要（识别情况 + 异常报告），再输出 CSV
7. **先看摘要**，确认识别正确、异常处理合理
8. **复制 CSV 部分**（以"量表名称"开头的行开始，到内容结束）
9. 打开后台 → 「量表管理」→「导入量表」→「CSV 格式」→ 粘贴到文本框
10. 点「解析预览」检查 → 确认后点「确认导入」

## 常见问题

**Q1：AI 没有生成 .csv 文件，而是输出了文字？**
最常见的问题。解决方法：
- **补充提示**：`请生成一个 .csv 文件供我下载，不要在聊天框里输出文字。使用你的文件生成工具（File/Code Interpreter/Artifact）创建文件。`
- **ChatGPT**：通常用 Code Interpreter / Advanced Data Analysis 自动生成文件下载
- **Claude**：使用 Artifacts 功能，或提示 `生成一个 csv 文件` 会触发文件生成
- **通义千问**：提示 `请以文件形式输出 CSV`，通常会生成附件下载
- 如果 AI 确实无法生成文件，它会退而输出 CSV 文本代码块，你复制内容另存为 .csv 即可

**Q2：AI 输出了 Markdown 代码块（三个反引号）？**
复制时只取内部内容，去掉首尾的 ` ```csv ` 和 ` ``` ` 两行。

**Q3：文件里有多张量表，AI 只输出了一张？**
在补充信息中明确：`文件中包含 N 张量表，请全部提取，位置以"附1："/"附2："标识`。

**Q4：导入后题目数量不对？**
检查是否合并了多个版本的题目（含简版/完整版的手册常见），以及后续题目行基本信息列是否误填（应留空）。

**Q5：导入后选项分值全为 0？**
AI 漏填了分值列，检查"选项X分值"是否有数字。注意起始分值：有的量表从 1 开始（1/2/3/4），有的从 0 开始（0/1/2/3）。

**Q6：反向计分题分值没翻转？**
补充提示：`反向计分题分值必须翻转，公式 = 最小值 + 最大值 - 原始分值`。检查 CSV 中反向题的 A 选项分值是否是最大值。

**Q7：CSV 列数与题目选项数不匹配？**
正常：列数以最多选项数为准，少的留空。若 AI 列数不够，提示：`最多有 N 个选项，表头须有 N 列选项列`。

**Q8：描述或指导语含逗号导致 CSV 错位？**
中文逗号（，）无需处理。英文逗号（,）需用双引号括起：`"内容, 含逗号"`。若 AI 未处理，手动在字段两端加 `"` 即可。

**Q9：量表分类填错了？**
后台「量表管理」→ 编辑该量表 → 修改分类。可选值：`anxiety` / `depression` / `personality` / `comprehensive` / `stress` / `unknown`。

**Q10：手册题目有缺失/编号不连续，怎么办？**
AI 会在摘要中报告异常（缺失/重复/错乱/残缺等）。先确认处理方式合理，再决定是否导入。缺失的题目不会被编造内容，需要你手动补充。

**Q11：AI 给所属分类填了 family、children 等不支持的值？**
手动修改即可。系统只支持 6 个分类：anxiety / depression / personality / comprehensive / stress / unknown。

**Q12：AI 改写了题目内容（用自己的话重述了题目）？**
这是严重的质量问题。补充提示：`题目内容必须逐字照搬手册原文，不得改写、意译、简化或替换措辞。每道题的文字必须和原文完全一致。` 如果 AI 仍然改写，可能是手册原文被你分开发送导致 AI 只看到了部分原文——确保提示词和手册原文一次性粘贴发送。
