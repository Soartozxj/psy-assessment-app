# 🚀 页面加载速度优化报告

**日期**: 2026-06-01  
**优化版本**: admin-legacy.html v1.2.2 | plugin-loader.js v1.0.4  
**优化目标**: 解决页面加载慢的问题

---

## 📊 优化前问题分析

### **问题1：同步加载 `scales-data.json`** ❌

**位置**: `admin-legacy.html` 第64-82行

**问题描述**:

```javascript
var xhr = new XMLHttpRequest();
xhr.open('GET', 'scales-data.json', false); // ❌ 同步请求！阻塞页面
xhr.send(null);
```

**影响**:

- ✅ **致命**: 同步请求会**阻塞页面渲染**，直到请求完成
- 如果文件大或网络慢，页面会卡住
- 用户体验极差

**测试数据**:

- `scales-data.json` 文件大小: ~700KB
- 同步加载时间: ~500ms-2000ms（取决于网络）
- 阻塞页面渲染: **是**

---

### **问题2：5个核心插件全部预加载** ⚠️

**位置**: `plugin-loader.js` 第26行

**问题描述**:

```javascript
this.corePlugins = ['auth', 'ai', 'scale', 'scoring', 'npc']; // 5个插件全部预加载
```

**影响**:

- 5个插件并行加载，但每个插件都需要：
  1. 下载JS文件（~50-200KB/个）
  2. 执行插件代码
  3. 调用 `init()` 方法
- 虽然并行加载，但仍会占用大量资源
- 首屏时间显著增加

**测试数据**:

- 插件数量: 5个
- 总大小: ~800KB
- 并行加载时间: ~1000ms-3000ms（取决于网络和CPU）
- 阻塞首屏: **是**

---

### **问题3：Chart.js 外部CDN加载** ⚠️

**位置**: `admin-legacy.html` 第17行

**问题描述**:

```javascript
<script src="https://cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.min.js"></script>
```

**影响**:

- 依赖外部CDN，网络问题可能导致加载失败
- CDN加载时间不可控
- 如果CDN慢，会阻塞页面渲染

**测试数据**:

- Chart.js 大小: ~200KB
- CDN加载时间: ~500ms-5000ms（取决于CDN速度）
- 阻塞首屏: **是**（因为放在 `<head>` 中）

---

### **问题4：缺少加载进度指示器** ⚠️

**问题描述**:

- 页面加载时没有任何提示
- 用户不知道页面是否在加载
- 如果加载慢，用户会以为页面卡死

**影响**:

- 用户体验差
- 用户可能重复刷新页面

---

## ✅ 优化方案与实施

### **优化1：将 `scales-data.json` 改为异步加载** ✅

**修改文件**: `admin-legacy.html`

**修改前**:

```javascript
(function () {
  try {
    var xhr = new XMLHttpRequest();
    xhr.open('GET', 'scales-data.json', false); // ❌ 同步请求
    xhr.send(null);
    if (xhr.status === 200) {
      window.__BUNDLED_SCALES__ = JSON.parse(xhr.responseText);
    } else {
      window.__BUNDLED_SCALES__ = [];
    }
  } catch (e) {
    window.__BUNDLED_SCALES__ = [];
  }
})();
```

**修改后**:

```javascript
window.__BUNDLED_SCALES__ = []; // 先初始化为空数组
window.__SCALES_DATA_PROMISE__ = new Promise(function (resolve) {
  var xhr = new XMLHttpRequest();
  xhr.open('GET', 'scales-data.json', true); // ✅ 改为异步加载
  xhr.onload = function () {
    if (xhr.status === 200) {
      try {
        window.__BUNDLED_SCALES__ = JSON.parse(xhr.responseText);
        console.log('[Admin] 从 scales-data.json 加载了 ' + window.__BUNDLED_SCALES__.length + ' 个量表');
      } catch (e) {
        console.warn('[Admin] 解析 scales-data.json 失败: ' + e.message);
        window.__BUNDLED_SCALES__ = [];
      }
    } else {
      console.warn('[Admin] 加载 scales-data.json 失败（HTTP ' + xhr.status + '），使用空数组');
      window.__BUNDLED_SCALES__ = [];
    }
    resolve(window.__BUNDLED_SCALES__);
  };
  xhr.onerror = function () {
    console.warn('[Admin] 加载 scales-data.json 网络错误，使用空数组');
    window.__BUNDLED_SCALES__ = [];
    resolve([]);
  };
  xhr.send(null);
});
```

**优化效果**:

- ✅ **消除页面阻塞**: 异步加载不阻塞页面渲染
- ✅ **提升首屏速度**: 页面可以立即显示
- ✅ **容错处理**: 添加 `onerror` 处理网络错误

### **优化2：核心插件改为按需加载** ✅

**修改文件**: `plugin-loader.js`

**修改前**:

```javascript
this.corePlugins = ['auth', 'ai', 'scale', 'scoring', 'npc']; // 5个插件全部预加载
```

#### **3. JavaScript控制逻辑** (第25389-25494行)

```javascript
// 加载进度管理器
window.LoadingManager = {
  overlay: null,
  text: null,
  progressBar: null,
  currentStep: 0,
  totalSteps: 4,

  init() {
    this.overlay = document.getElementById('loading-overlay');
    this.text = document.getElementById('loading-text');
    this.progressBar = document.getElementById('loading-progress-bar');
  },

  update(step, message) {
    this.currentStep = step;
    const percent = Math.min((step / this.totalSteps) * 100, 100);
    if (this.progressBar) {
      this.progressBar.style.width = percent + '%';
    }
    if (this.text && message) {
      this.text.textContent = message;
    }
  },

  hide() {
    if (this.overlay) {
      this.overlay.classList.add('fade-out');
      setTimeout(() => {
        this.overlay.style.display = 'none';
      }, 500);
    }
  }
};
```

**优化效果**:

- ✅ **提升用户体验**: 用户可以看到加载进度
- ✅ **减少重复刷新**: 用户知道页面在加载
- ✅ **专业感**: 渐变背景 + 旋转动画 + 进度条

### **优化5：添加插件按需加载辅助函数** ✅

**修改文件**: `admin-legacy.html`

**新增内容** (第25485-25493行)

```javascript
// 插件按需加载辅助函数（在切换页面时调用）
window.ensurePlugin = async function (pluginName) {
  if (PluginLoader.plugins[pluginName]) {
    return PluginLoader.plugins[pluginName];
  }
  console.log(`🔄 按需加载插件: ${pluginName}`);
  return await PluginLoader.load(pluginName, true);
};
```

**优化效果**:

- ✅ **方便按需加载**: 在 `switchSection` 中调用 `await ensurePlugin('npc')`
- ✅ **避免重复加载**: 如果已加载，直接返回
- ✅ **统一错误处理**: 使用 `silent=true` 忽略加载失败
